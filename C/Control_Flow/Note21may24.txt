...........
    if (expression)
        ..statement..

4*Switch*
.........
    switch (expression) {
        case const-expressions: statement
        case const-expressions: statement

        default: statement
    }

5*For-while-loop*
.................
    while (expression){
        statement
    }

    for (exp1; exp2; exp3){
        statement
    }

    for loop is basically a spacial while loop where,

    exp1;
    while (exp2){
        statement;
        exp3;
    }

6*Do-while-loop*
................

    do 
        statement
    while (expression)

    The statement is executed then expression is evaluated. If it is true, statement is evaluated again, and so on.
